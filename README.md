# guess_game_with_dict
# 🧠 Guess the Number (Dictionary Edition)

A structured version of the number guessing game using Python dictionaries to store game state.

## 🚀 How it works

- The game picks a random number between 1 and 10
- You have 3 attempts to guess it
- All game state is stored in a Python dictionary
- Win or lose, you get feedback!

## 🛠 Technologies

- Python 3
- `random` module
- `dict` for managing state

## ▶️ Run the game

```bash
python3 guess_game_with_dict.py
🎯 Guess a number between 1 and 10: 3  
❌ Wrong guess. Try again.  
🎯 Guess a number between 1 and 10: 7  
🎉 Congratulations! You guessed the number!
